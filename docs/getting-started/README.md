# `docs/getting-started`

该目录面向第一次使用项目的用户，优先解决三个问题：

- 这个项目现在能做什么
- 我该怎么把它跑起来
- 什么时候需要配置真实 `DeepSeek`

## 建议阅读顺序

1. `quick-start.md`
2. `real-provider.md`
3. `faq.md`

## 当前官方口径

- 平台：`Windows + PowerShell`
- 部署方式：本地单用户手动部署
- 默认体验：未配置 API Key 也可通过 deterministic fallback 完成首次运行
- 数据存储：`SQLite`

更深的配置、诊断和回归说明见 `docs/operations/`。
