# 术语表

## 核心术语

### 小说智能打分系统

指面向小说文本输入，基于 `LLM as Judge` 输出结构化评分结果的系统。

### 稿件

指用户提交用于分析和评分的文本内容，可能是开篇、章节、大纲或其它相关文本。

### 评测任务

指一次从输入接收到结果返回的独立评测执行过程。

### Prompt

指用于指导模型执行评分、提取或校准任务的受控文本资产。

### Rubric

指用于定义评分维度、子维度、评分锚点、证据要求、风险信号与聚合口径的正式评判规则骨架。

### 阶段契约

指评分流水线在中间阶段产出的结构化 JSON 对象，用于连接预检查、证据抽取、原子评分、一致性校验、聚合与仲裁等环节。

### Provider

指承载大模型能力的供应商或模型接入来源。

### Schema

指输入或输出结构的正式约束定义。

### Evals

指用于验证结构稳定性、评分一致性和版本回归情况的评测体系。

### 基线

指历史可接受输出结果或评测结果，用于后续版本比较。
