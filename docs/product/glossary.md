# 术语表

## 核心术语

### 小说智能打分系统

指面向小说文本输入，基于 `LLM as Judge` 输出结构化评分结果的系统。

### 稿件

指用户提交用于分析和评分的文本内容，可能是开篇、章节、大纲或其它相关文本。

### 评分任务

指一次从输入接收到结果返回的独立执行过程。

### Prompt

指用于指导模型执行评分、提取或校准任务的受控文本资产。

### Provider

指承载大模型能力的供应商或模型接入来源。

### Schema

指输入或输出结构的正式约束定义。

### Evals

指用于验证结构稳定性、评分一致性和版本回归情况的评测体系。

### 基线

指历史可接受输出结果或评测结果，用于后续版本比较。
