# ADR-003：`output/playwright` 定义为研究产物区

## 状态

已接受。

## 背景

当前仓库已存在 `output/playwright/` 目录及若干抓取与分析产物。这些文件有研究价值，但并不适合作为正式系统源码的基础。

## 决策

- `output/playwright/` 永久定义为研究/抓取产物区
- 该目录中的文件不纳入 `apps/` 或 `packages/`
- 正式业务逻辑不得直接依赖该目录
- 如需吸收其中信息，应转写为正式文档或正式领域定义

## 结果

- 正式工程与研究产物边界清晰
- 避免历史抓取文件污染长期代码结构
- 保留分析价值，同时降低维护噪音
