# `apps/worker`

该目录用于承载异步任务执行层。

## 典型职责

- 处理长文本评分任务
- 运行批量回归评测
- 执行离线分析、报告生成与重试任务
- 隔离耗时操作，避免接口层承担长时工作

## 实现约束

- `apps/worker` 是后台执行入口，不是业务语义真源
- 应复用 `packages/application/` 中的用例编排
- 多阶段评分执行应优先复用 `PocketFlow` 编排抽象
- 不应独立定义第二套任务状态、错误语义或 Prompt 选择规则

## Phase 1 说明

- 当前阶段先冻结目录边界与职责
- 当前不预设复杂任务队列或分布式调度系统
- 本地部署场景下可先采用轻量、本机可运行的执行方式
