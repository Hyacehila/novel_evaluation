# `apps/api`

该目录用于承载服务端接口层。

## 职责

- 接收并校验外部请求
- 编排评分任务与结果返回
- 调用应用层能力完成 Prompt 选择、Provider 调用、Schema 校验
- 对前端提供稳定接口与错误语义

## 设计原则

- 只做接口边界与流程编排，不承载底层 Provider 细节
- 只依赖正式领域模型与契约，不直接依赖研究产物
- 所有输出必须符合严格 JSON 契约

## 当前初始化状态

- 已在本目录建立 Python `uv` 工程基线
- Python 版本基线为 `3.13`
- 当前仅完成结构与环境初始化，尚未开始正式后端业务开发
- 后续执行 Python 命令统一使用 `uv run`
