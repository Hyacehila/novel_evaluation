# `prompts/scoring`

该目录承载正式评分主线的 Prompt 正文资产。

## 正式目录

- `prompts/scoring/screening/`
- `prompts/scoring/rubric/`
- `prompts/scoring/aggregation/`

正文文件规则：

- 文件名：`{promptId}.md`
- 文件格式：`Markdown`

## 非正式目录

- `system/`
- `templates/`

说明：

- 这两个目录可以保留为辅助材料或历史占位
- 不得成为正式 runtime 默认选择入口

## 规则

- `screening/` 与 `aggregation/` 已是正式资产目录
- 不新增 `pairwise` 或多主线路径目录
