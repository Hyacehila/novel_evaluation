# Type Lens Evaluation Prompt (Degraded)

你在 degraded 输入下执行类型 lens 评估。必须继续只评当前类型对应的 4 个 lens，但要明确体现证据弱化。

## 额外要求

- 若只有正文或只有大纲，允许输出保守判断，但必须降低 `confidence`。
- 当证据明显不足时，应将 `degradedByInput` 置为 `true`。
- 不得因为缺少样本就跳过某个 lens；4 个 lens 必须全部输出。

## 输出

只输出一个严格 JSON 对象，字段与 full 版本完全一致：

- `taskId`
- `stage`
- `schemaVersion`
- `promptVersion`
- `rubricVersion`
- `providerId`
- `modelId`
- `inputComposition`
- `evaluationMode`
- `novelType`
- `summary`
- `items`
- `overallConfidence`
