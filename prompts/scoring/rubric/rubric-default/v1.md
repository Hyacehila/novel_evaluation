# Rubric Prompt (Full)

你负责 `rubric_evaluation` 阶段。用户消息会给出 `requestedAxes`，你只评价这些轴，不要补写其他轴。

## 8 轴定义

- `hookRetention`
- `serialMomentum`
- `characterDrive`
- `narrativeControl`
- `pacingPayoff`
- `settingDifferentiation`
- `platformFit`
- `commercialPotential`

## 输出

只返回一个 JSON object。字段必须严格限制为：

- `taskId`
- `stage`
- `schemaVersion`
- `promptVersion`
- `rubricVersion`
- `providerId`
- `modelId`
- `inputComposition`
- `evaluationMode`
- `requestedAxes`
- `items`
- `axisSummaries`
- `missingRequiredAxes`
- `riskTags`
- `overallConfidence`

## `items` 结构

`items` 必须与 `requestedAxes` 一一对应，且只覆盖 `requestedAxes`。每个 item 必须包含：

- `evaluationId`
- `axisId`
- `scoreBand`
- `reason`
- `evidenceRefs`
- `confidence`
- `riskTags`
- `blockingSignals`
- `degradedByInput`

### `scoreBand`

只能使用字符串：

- `"0"`
- `"1"`
- `"2"`
- `"3"`
- `"4"`

### `evidenceRefs`

至少 1 条。每条必须包含：

- `sourceType`: `chapters` / `outline` / `cross_input`
- `sourceSpan`: 对象
- `excerpt`: 引文或摘要摘录
- `observationType`
- `evidenceNote`
- `confidence`

## 其余字段要求

- `axisSummaries`：对象，键必须与 `requestedAxes` 完全一致
- `missingRequiredAxes`：只能引用 `requestedAxes` 中无法负责任判断的轴；在 full 模式通常应为空
- `riskTags`：只能使用正式枚举值
- `overallConfidence`：`0` 到 `1`
- `degradedByInput`：full 模式通常为 `false`

## 强约束

- 不要输出 `requestedAxes` 之外的轴
- 不要缺 item
- 不要输出额外字段
- 不要输出旧骨架字段或兼容字段
